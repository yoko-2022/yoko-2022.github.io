Works

家計簿アプリ

機能
収支管理、項目追加・削除、月間レポート、グラフ表示
デザインについて
収入は青、支出は赤としてわかりやすくしました。グラフを表示させて視覚的にわかりやすくしました。
コーディングについて
  • form要素の入力値をReactを使ってデータをトラッキングし、変更点だけを再描画させることで表示の高速化を行いました。
  • 項目の値をデータベースに保存、保存したデータを取り出しました。その際、LaravelのMVCモデルを使用してデータベースとやり取りを行いました。
  • データの削除は削除以外のデータをmap関数で展開させることで解決しました。
  • React-chart.jsを使ってグラフ表示を行いました。
  • コーディングの補助としてChatGPTを使用しましたが、
ツール
React,Laravel,MySQL,XAMPP,Composer,CSS,React-chart.js

The Active Kitchen

サイトの内容
ヘルシー志向のレストランサイト
ターゲット
20代、30代の男女とその家族
デザインについて
オレンジと黄緑を基調としたフレッシュなデザインで、コンテンツの境目に曲線、斜め線などのアレンジをしました。
コーディングについて
  • スマートフォンサイズでは上にスクロールしたときにだけナビゲーションが出てくるようにしました。
  • トップ画像はヒーローヘッダーにして、さらにclip-pathを使って画像を斜めに切り取りました。
  • cssだけでスライドショーや吹き出しを作成しました。
  • target="_blank"の脆弱性対策としてrel="noopener"を付けました。
  • Googleカレンダー、Twitterを埋め込みました。
ツール
HTML,CSS,jQuery,Googleカレンダー,twitter-api

Skill

  • HTML

    HTML Living Standardの仕様に基づき、セマンティックなHTMLを書いています。
    video要素による動画の埋め込み、picture要素によるアートディレクションを学びました。
    form要素のautocomplete属性を使うことで入力補助を簡単に扱い、EFOを考慮したわかりやすいフォームを作成できます。
    わかりやすいクラス名を意識しています。

  • CSS

    アニメーションやレスポンシブ対応、flexboxやgridレイアウト、フルードイメージやでobject-fitで自由なレイアウトができます。
    『Can I use』を使って新しいcssが各ブラウザーに対応しているかを調べて、場合によってはベンダーブレフィックスを使用しています。
    vh、vwや％、remなどを使用する事で、様々なレイアウトをマルチデバイスに対応できます。

  • SASS

    SCSS記法を使ってのSASSを書くことができます。ネスト、変数、mixin、extendなどのsassの機能を使用できます。
    コンパイラーを使ってcssにコンパイルすることができます。
    CSSよりSASSが好きです。

  • JavaScript

    DOMの操作を行うことができます。JavaScriptの基本的な文法やデータ型、オブジェクト指向プログラミングなどの理解があります。
    Ajaxで読み込むデータを外部ファイル化し、非同期通信する事で必要な物を必要な分だけ必要な時に読み込むというファイル操作ができます。
    三項演算子、テンプレート文字列、スプレッド構文、map・filter関数などの関数を使用して見やすいコードを書くようにしています。

  • jQuery

    様々なプラグインの使用、クリックやスクロールイベントの処理ができます。クラス名の追加・削除や絵each関数の使用などができます。
    トラバースを利用して操作したい要素から見てどのような関係性にある要素なのかを考え、指定することができます。

  • React

    SPA（シングル・ページ・アプリケーション）の作成方法を学びました。
    useState、useEffectなどのHooksやAPIを用いることができます。

  • PHP・WordPress

    XAMPPを使った仮想サーバーの構築ができます。
    get/postを使ったデータ通信、データベースとの接続およびデータの挿入や検索ができます。
    お問い合わせフォームの実装やWordPressのテーマ作成ができます。
    WordPressのオリジナルテーマを作成することができます。また本番環境へデプロイすることができます。

  • Laravel

    LaravelのMVCモデル、Eloquent ORMを用いたデータベース操作ができます。

  • Photoshop/ Illustrator/ Figma

    画像の切り抜き、画像トレース、レイヤーマスク、クリッピングマスクの使用ができます。
    アートボードを使用してカンプやバナーを複数展開させることができます。
    アセット機能を使った画像の書き出しができます。
    調整フィルターを使った画像補正ができます。
    フィルター機能を使用した画像加工ができます。
